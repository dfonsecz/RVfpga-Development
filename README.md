# RVfpga Development

Este repositorio contiene algunos ejercicios resueltos de las guías de [RVfpga: Understanding Computer Architecture](https://www.imaginationtech.com/teaching/). Estos ejercicios fueron desarrollados con el objetivo de profundizar en el manejo del sistema **RISC-V** y en la implementación de programas sobre FPGA, así como en la manipulación del código Verilog para la integración de nuevos periféricos.

A través de estos laboratorios, se busca reforzar la comprensión de la arquitectura de computadores y la interacción entre el hardware y el software.
